# JPMorgan Chase & Co. (JPM) — Equity Research Initiation

A professional, single-page equity research report on **JPMorgan Chase & Co. (NYSE: JPM)**, built entirely using **HTML and CSS**.

The project replicates the structure and presentation style of institutional sell-side equity research reports, featuring investment recommendations, financial analysis, valuation methodologies, and risk assessment.

## Features

- Professional equity research report layout
- Sticky sidebar navigation
- Responsive design for desktop and mobile
- Investment summary and recommendation
- Company overview and business segments
- Industry and peer comparison
- Investment thesis and growth drivers
- Financial projections and analysis
- Valuation using:
  - Price-to-Earnings (P/E)
  - Price-to-Tangible Book Value (P/TBV)
- Key risks and catalysts
- Institutional-style typography and formatting

## Technologies Used

- HTML5
- CSS3
- Responsive Web Design
- SVG for financial visualizations

## Running the Project

Simply open the HTML file in your browser:

```bash
open jpmc_equity_research.html
```

or double-click:

```text
jpmc_equity_research.html
```

No installation, dependencies, or build tools are required.
